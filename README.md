# Pat Wakeem - Professional Portfolio

This repository hosts the static portfolio website of **Pat Wakeem**, Staff Software Engineer / Software Architect specializing in backend architecture, cloud infrastructure, and technical leadership on the AWS platform.

## Features

- **Responsive Single-Page Layout**: Works seamlessly across mobile, tablet, and desktop viewports.
- **Premium Aesthetics**: High-end dark/light theme switching with local storage persistence.
- **Interactive Experience Timeline**: Expandable career highlights detailing roles at Tink (A Visa Solution) and Apica Systems.
- **Zero Dependencies**: Built with pure Vanilla HTML5, CSS3, and JavaScript, meaning zero build overhead and instant load times.
- **Under Construction Badge**: Displayed to highlight that the site is currently being rebuilt.

## Deployment

The website is hosted on GitHub Pages and can be accessed at:
[https://patwakeem.github.io](https://patwakeem.github.io)

Since it is built with pure static files, deployment is as simple as pushing changes directly to the `master` branch.

## Local Development

You can preview the website locally by opening `index.html` in your web browser or serving it using a lightweight local web server.

### Using Python (Recommended)
Run the following command in your terminal:
```bash
python3 -m http.server 8000
```
Then open `http://localhost:8000` in your browser.
